# Notulee Community

Welcome to the community space for **[Notulee](https://notulee.com)** — a keyboard-driven scratchpad with superpowers: plain-text notes with live math, currency, units, reactive variables, checklists and instant search. Private and local, no account required.

This repository contains **no app source code** — it exists purely so you can report bugs, request features, ask questions, and share tips.

## 🐛 Filing a good bug report

Notulee runs entirely in your browser, so the details that help us most are:

- **What you did**, step by step
- **What you expected** vs. **what actually happened**
- Your **browser + OS** (e.g. Firefox 128 / Windows 11)
- Roughly **when you last loaded the app** (it updates often)
- Any **console errors** (press <kbd>F12</kbd> → Console) or screenshots

The bug-report form will prompt you for each of these.
